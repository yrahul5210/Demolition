All Git Command

git switch -C Bipin-Branch  // branch created and switch to creation branch
git branch   // to check all created branch
git checkout Bipin-Branch  // to switch from main branch to Bipin-Branch

git add .
git commit -m "Changes in Git Folder"
git push -u origin Bipin-Branch   // all changes has been push to Bipin-Branch

To Merge with main Branch
git checkout main //  first switch to main Branch 
git fetch //  first Fetch
then you will see all branch that's has been created
git Merge Bipin-Branch  // git Merge Bipin-Branch to Merge
git push -u origin main // now push all code to main branch

git pull // (fetch + merge)
